## Plugin Overview

Mulmocast presentation plugin for GUI Chat applications. Create and display multimedia presentations.

## Common Guidelines

For standard plugin development guidelines, see:
https://github.com/receptron/GUIChatPluginTemplate/blob/main/CLAUDE.md

## Plugin-Specific Notes

### Features
- Multimedia presentation display
- Integration with Mulmocast format
- Audio/video content support

### Dependencies
- `gui-chat-protocol`: Core protocol for GUI Chat plugins
